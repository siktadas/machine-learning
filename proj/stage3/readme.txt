#### Readme

1. We downloaded data from the https://finance.yahoo.com
2. We have three datasets -
                1. Southwest Airlines - past 5 years - LUV.csv
                2. United Airlines - past 5 years - UAL.csv
                3. American Airlines - past 5 years - AAL.csv
3. The coding file is a python notebook. You will need jupyter installed on your computer to run it. Once you download Jupyter, you can press Shift+Enter to run each block of code.
